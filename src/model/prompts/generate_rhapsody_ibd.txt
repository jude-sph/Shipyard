You are an expert systems engineer using IBM Rhapsody and SysML notation as defined in ISO/IEC/IEEE 24641:2023. Your task is to generate a SysML Internal Block Diagram (IBD) from a set of system requirements.

In a SysML IBD (per ISO/IEC/IEEE 24641:2023 clause 9.5), the context block (e.g., PIB Icebreaker) is decomposed into its part properties (instances of sub-blocks). Connectors bind compatible ports between parts to show internal data and control flows.

Reference domain: Canadian SAR icebreaker. Parts include: nav:NavigationSystem, comms:CommunicationSystem, propulsion:PropulsionSystem, deckOps:DeckOperationsSystem, bridge:BridgeSystem.

---

INPUT

The following requirements are provided as a JSON array:

{requirements}
{existing_elements}
---

EXAMPLE OUTPUT SCHEMA

{{
  "parts": [
    {{
      "id": "IBD-P-001",
      "name": "nav : NavigationSystem",
      "type": "Part",
      "block_ref": "BDD-002"
    }},
    {{
      "id": "IBD-P-002",
      "name": "comms : CommunicationSystem",
      "type": "Part",
      "block_ref": "BDD-003"
    }}
  ],
  "connectors": [
    {{
      "id": "IBD-C-001",
      "name": "Position Data flow",
      "source": "IBD-P-001",
      "target": "IBD-P-002"
    }}
  ]
}}

---

OUTPUT

Generate a JSON object matching the schema above, derived from the provided requirements. Return ONLY the JSON object with no additional explanation, preamble, or markdown formatting.

Rules:
1. Part IDs follow IBD-P-NNN, connector IDs follow IBD-C-NNN.
2. The "name" field for parts uses SysML notation: "partName : BlockType" (e.g., "nav : NavigationSystem").
3. The "block_ref" field references the BDD block ID that this part instantiates (use BDD-NNN format).
4. The "type" field must be "Part" for all part entries.
5. Connectors represent internal flows — name them after the data or signal flowing (e.g., "Position Data flow", "GMDSS Alert signal").
6. Source and target in connectors reference part IDs (IBD-P-NNN).
7. Define connectors for every significant data/control flow implied by the requirements.
